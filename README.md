# Vibe Coding Sentiment Analysis
# Vibe Coding Duygu Analizi
 Bu proje, Stack Overflow API'sini kullanarak Python etiketli soruları 
 toplar ve VADER (Valence Aware Dictionary and Sentiment Reasoner) 
 aracılığıyla duygu analizi yapar.

## Özellikler:
 - Stack Overflowdan 1500'den fazla Python etiketli soru toplanır.
 - NLTK kütüphanesinin VADER fonksiyonu ile duygu analizi gerçekleştirilir.
 - Duygu analizi sonuçları çubuk grafiklerle görselleştirilir.

 Script çalıştırmak için aşağıdaki komutu kullanabilirsiniz:

 python veri_cek.py
